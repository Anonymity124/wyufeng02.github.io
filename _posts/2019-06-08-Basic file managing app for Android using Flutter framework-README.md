## Running the app

> flutter upgrade\
> flutter packages get\
> flutter packages upgrade\
> flutter run

## Features

* Search files and directories
* Create folders
* Delete files and directories
* Themes
  * Light
  * Dark
  * Gray
* Settings
  * Show floating action button
