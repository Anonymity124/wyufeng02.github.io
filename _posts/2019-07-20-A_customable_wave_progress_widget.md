# 如何使用
  1.将其添加到包的pubspec.yaml文件中：

````dart
dependencies:
  wave_progress_widget: ^0.0.1
  ````
  2.将此导入添加到您的dart文件：

````dart
import 'package:wave_progress_widget/wave_progress.dart';
  ````
3.只需在任何地方添加WaveProgress即可！
  你也可以自定义它的边框和填充颜色！
 
 ````dart
 WaveProgress(180.0, Colors.blue, Colors.blueAccent, _progress)
  ````
